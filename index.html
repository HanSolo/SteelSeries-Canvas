SteelSeries-Canvas by HanSolo

The HTML5 Canvas port of the SteelSeries component library. Mainly contains gauges.

Dependencies

You need tween.js (which is also in the repository) to use the lib. Tween.js is a tweening library based on the tweening functions from R. Penner.
